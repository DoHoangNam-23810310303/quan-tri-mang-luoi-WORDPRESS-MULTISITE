A lightweight WordPress multisite plugin for Super Admin to view quick stats of all network sites.

== Features ==
- Network Activate supported
- Network Admin menu page
- Table columns: Site ID, Site Name, Post Count, Storage Used, Latest Post Date
- Uses get_sites(), switch_to_blog(), restore_current_blog()

== Installation ==
1. Upload folder `network-site-stats` to `wp-content/plugins/`
2. Go to Network Admin > Plugins
3. Click `Network Activate` on Network Site Stats
4. Open Network Admin > Site Stats
